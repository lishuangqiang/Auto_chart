# Auto_BI数据分析平台

#### 介绍
利用AI能力自主解析excle数据文件，生成前端图表代码和配套数据解析。
配套异步方法和同步方法对图表进行处理

#### 安装教程

1.  更新Mysql和redis地址，即可运行
####注意点
1.本项目仅包含后端，无前端，如果有感兴趣的同学，可以自己写前端和其绑定
